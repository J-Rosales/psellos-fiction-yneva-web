# Desktop Stable Release Checklist

## Policy Checks

- Specs and ADRs are synchronized.
- Layer hard-filter behavior is documented and implemented.
- URL-state/filter policy is consistent with UX spec.
- Graph and map side-panel patterns are aligned.

## Build and Packaging Checks

- `npm run build` succeeds.
- No contract blocker errors from artifact audit.
- Release notes include gate report summary.

## Post-Stable Note

- Mobile and accessibility hardening are scheduled for post-stable phase.
